מדריך אופרטורים לחיפוש בטוויטר

חיפוש בטוויטר יודע הרבה יותר ממה שתיבת החיפוש מראה. מאחורי הקלעים יש שפת שאילתות שלמה: אפשר לבקש רק ציוצים עם יותר מ-100 לייקים, רק כאלה שכוללים סרטון, רק מחשבון מסוים, ורק מהחודש האחרון — והכול בשורה אחת.

העמוד הזה מרכז את כל האופרטורים שטוויטר מכיר, עם הסבר קצר ודוגמה לכל אחד. אם לא בא לכם לזכור אותם, הכלי שלנו בונה את השאילתה במקומכם — אבל שווה להכיר, כי ברגע שמבינים את ההיגיון אפשר לכתוב חיפושים שרוב המשתמשים לא יודעים שאפשריים.

שילוב אופרטורים: רווח בין שני תנאים פירושו "וגם". כלומר קיפוד min_faves:100 יחזיר ציוצים שמכילים את המילה קיפוד וגם קיבלו לפחות 100 לייקים.

מילים וביטויים

אופרטורמשמעותדוגמה
cat dog שני הביטויים חייבים להופיע (זו ברירת המחדל — רווח = וגם). קיפוד כיפה
"exact phrase" מחפש את הרצף המדויק, במרכאות. "קיפוד עם כיפה"
cat OR dog לפחות אחת מהמילים. OR חייב להיות באותיות גדולות. קיפוד OR חתול
-word מחריג ציוצים שמכילים את המילה. קיפוד -חתול
#hashtag ציוצים עם ההאשטג. #קיפודים
(a OR b) c סוגריים מקבצים תנאים — שימושי לשילובים. (קיפוד OR חתול) חמוד

Twitter search operators guide

Twitter search understands far more than the search box suggests. Behind it sits a full query language: you can ask for only tweets with more than 100 likes, only ones containing video, only from a given account, and only from the last month — all in a single line.

This page collects every operator Twitter understands, with a short explanation and an example for each. If you'd rather not memorise them, our tool builds the query for you — but they're worth knowing, because once the logic clicks you can write searches most people don't realise are possible.

Combining operators: a space between two conditions means "AND". So hedgehog min_faves:100 returns tweets containing the word hedgehog that also got at least 100 likes.

Words & phrases

OperatorMeaningExample
cat dog Both words must appear (space = AND, the default). hedgehog hat
"exact phrase" Matches the exact sequence, in quotes. "hedgehog with a hat"
cat OR dog At least one of the words. OR must be uppercase. hedgehog OR cat
-word Excludes tweets containing the word. hedgehog -cat
#hashtag Tweets carrying that hashtag. #hedgehogs
(a OR b) c Parentheses group conditions for combinations. (hedgehog OR cat) cute

Accounts

OperatorMeaningExample
from: Tweets from this user. Comma = multiple users. from:eitanshimoni
to: Tweets in reply to this user to:eitanshimoni
@ Tweets mentioning this user @eitanshimoni
list: Tweets from members of a given list. list:123456

Dates

OperatorMeaningExample
since: Tweets on or after this date since:2026-01-01
until: Tweets before this date until:2026-12-31
within_time: Tweets from the recent window (minutes/hours/days). within_time:2d

Engagement

OperatorMeaningExample
min_faves: Minimum number of likes min_faves:100
min_retweets: Minimum retweet count min_retweets:50
min_replies: Minimum reply count min_replies:10

Content type

OperatorMeaningExample
filter:links Only tweets with URLs hedgehog filter:links
filter:media Only tweets with any media hedgehog filter:media
filter:images Only tweets with images hedgehog filter:images
filter:videos Only tweets with videos hedgehog filter:videos
-filter:retweets Exclude all retweets hedgehog -filter:retweets
filter:replies Only replies. With a minus — excludes replies. -filter:replies

Language & advanced

OperatorMeaningExample
lang: Tweets written in this language lang:en
url: Tweets containing a specific URL url:"github.com"
near: Tweets near a place (support varies). near:"Tel Aviv"
